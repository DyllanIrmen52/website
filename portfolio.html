This is my pizza maker project

My project uses commands to put toppings onto the pizza. When the simulator starts, it will ask you what toppings you want. Once you answer, it will cook it in front of you while you watch.

When spreading the paste, it spins in a circular manner. When cutting the pizza, the pizza cutter moves in a diagonal line.

2.1.6 project

The 2.1.6 project was given to us to test our new understanding of data protection. We were tasked with diagnosing a data breach in a fish tank monitoring software, it was a result of a phishing email. This made us use the debugger to find the exact area where the damage of the data breach was. After finding the damage to the program, we used our understanding of the functionality of the code to repair it.

4.1.4 Projet

In 4.1.4 we had to analyze a simulation. We started by running the simulation and playing with the variables to determine their function/effect on the code to answer these 9 questions: How the model works, how the turtles are represented, how the patches are represented, how randomness is used, what exclusions the model has, what questions we had about the model, what suggestions we had for the model, what conclusions we had, and how we can apply it to the real world.
